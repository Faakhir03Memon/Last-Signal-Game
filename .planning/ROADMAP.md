# Roadmap - Last Signal

## Phase 1: Foundation (Current)
- **1.1 Setup:** Project structure, basic scene, player controller.
- **1.2 Tools:** Flashlight and Signal Scanner core logic.
- **1.3 Signal Detection:** Signal source prefab and proximity logic.

## Phase 2: Interactivity
- **2.1 Puzzle Engine:** UI-based frequency matching system.
- **2.2 Memory System:** Scene management for entering/exiting missions.
- **2.3 First Mission:** "The Last Call" - A narrative vignette.

## Phase 3: Survival & Resource
- **3.1 Battery System:** Consumption and recharging.
- **3.2 UI/HUD:** Battery levels, scanner feedback.
- **3.3 Soundscape:** Ambient wind, radio static, footstep sounds.

## Phase 4: World Expansion
- **4.1 Terrain:** Larger low-poly map (Phase 1 Map Design).
- **4.2 POIs:** Abandoned house, Tower, Server room.
- **4.3 Save System:** Persistent progress and story state.

## Phase 5: Danger (Expansion)
- **5.1 Enemies:** Drone AI (Patrolling).
- **5.2 Stealth:** Visibility/Noise detection.
- **5.3 Combat/Evasion:** Basic distraction tools.
