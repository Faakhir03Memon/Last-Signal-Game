# Requirements - MVP (Phase 1)

## 🎯 MVP Goal
Create a playable loop: 1 Map -> 1 Signal -> 1 Puzzle -> 1 Memory Mission.

## 🏃 Player Controller
- [ ] 3D Character movement (WASD + Mouse Look).
- [ ] Sprinting and Jumping.
- [ ] Interaction system (Raycast based).
- [ ] Flashlight toggle (F key).

## 📡 Signal System
- [ ] Scanner tool (handheld model or UI indicator).
- [ ] Proximity detection for signals (BEEP frequency increases near signal).
- [ ] Visual signal source (broken server or antenna).

## 🧩 Puzzle System
- [ ] Frequency Match Mini-game: Sync two sine waves.
- [ ] Success state: Unlocks Memory Mission.

## 🎬 Memory Mission
- [ ] Simple "Flashback" scene transition.
- [ ] Audio-visual storytelling (Narrative voiceover + static images/low-poly vignette).
- [ ] Return to main map after mission ends.

## 🔋 Resource System
- [ ] Battery meter for Flashlight and Scanner.
- [ ] Battery pickups in the environment.

## 🎨 Art & Environment
- [ ] Low-poly terrain (100x100m for MVP).
- [ ] Basic lighting (Atmospheric fog, Dark environment).
- [ ] 1 Abandoned structure.
